## 4. Revamp plan

### Phase 1 — Technical + on-page foundation (weeks 1–2)
- **Homepage retitle:** `Chiropractor in Cheras (Maluri), Kuala Lumpur | Persistence Chiropractic`. H1 must contain "Chiropractor in Cheras". Currently the site doesn't anchor itself to Cheras/Maluri strongly enough to rank for proximity terms.
- **Schema markup:** `Chiropractic`/`LocalBusiness` JSON-LD with NAP, geo coords (Maluri), opening hours, `aggregateRating` once reviews flow.
- **Fix the "chiro cheras" ranking page:** Google now ranks the homepage #16 for it (previously the thin `/book-now` page). Still no dedicated Cheras page — build a proper Cheras/Maluri landing page and internally link it sitewide to convert the homepage's #16 into a top-3.
- Core Web Vitals pass, mobile-first, clean URL structure, XML sitemap, image alt text with local modifiers.

### Phase 2 — Local SEO (the real "chiropractor near me" lever, weeks 1–4)
"Near me" + the local pack (which sits at position 1–3 of every Cheras SERP) are won off-site:
- **Google Business Profile**: complete category (Chiropractor), Maluri service area, weekly posts, Q&A seeded, photos.
- **Review engine**: systematic post-visit review asks. Local pack ranking ≈ reviews × proximity × relevance.
- **Citations**: Yelp MY, ClassPass (Connect Chiropractic ranks via ClassPass), Foursquare, Waze/Google Maps, TripAdvisor health, local MY directories — exact NAP match.

### Phase 3 — Site architecture: service + location pages (weeks 2–6)
```
/                          → Chiropractor in Cheras (Maluri), KL
/services/back-pain        /services/slipped-disc
/services/scoliosis        /services/sciatica
/services/neck-pain-posture /services/sports-injury
/locations/cheras-maluri   (canonical local page)
/locations/kuala-lumpur    /locations/kepong  /locations/bangsar
                           (only if genuinely serviceable — no doorway spam)
```
Each service page: symptoms → how chiro treats it → pricing/booking CTA → FAQ block with FAQ schema.

### Phase 4 — Content engine (ongoing, 2–4 posts/mo)
Copy the proven Ian The Chiro playbook, Malaysia-angle everything:
1. **"Tit tar vs chiropractic — what's the difference?"** (1,000 vol, SD 12 — easiest big win in this niche)
2. Muscle knots: causes & treatment (1,000 vol)
3. "Best chiropractors in KL" listicle — Excellence ranks #9 for "chiropractor malaysia kuala lumpur" with exactly this; do it better
4. Slipped disc treatment options in Malaysia
5. Office workers' back pain (KL commuter/desk angle)
6. Trigger points 101

### Phase 5 — Authority building (ongoing)
Gap to close: 32 → 50+ referring domains. Tactics: local health/lifestyle media (KL foodie-adjacent lifestyle blogs cover clinics — see kellysiewcooks precedent), MY health directories (erufucare.com lists clinics), sponsor local sports/run events, guest expertise quotes.
